üìò How to Use Ops Link

Getting Started

Ops Link is a powerful Discord bot that tracks game statistics directly from source APIs. Here's how to get started:

  • Invite the bot to your Discord server
  • Use /link-game to connect your gaming accounts
  • Use /stats to view your statistics
  • Statistics are cached for 5 minutes for optimal performance

Linking Gaming Accounts

To link your gaming accounts, use the following commands:

  • Roblox: Provide your Roblox username
  • Fortnite: Provide your Epic Games username
  • Call of Duty: Provide your Activision ID
  • PUBG: Provide your PUBG username and platform
  • Battlefield: Provide your Origin/EA username

üîß Troubleshooting

Common Issues

  • Stats not updating: Statistics are cached for 5 minutes. Wait and try again.
  • Can't link account: Make sure your username is correct and your profile is public.
  • Command not working: Ensure the bot has proper permissions in your server.
  • Bot offline: Check the status page for real-time bot status.

üëë Server Owner Guide

Setting Up Ops Link

As a server owner, you have additional features:

  • TXAdmin Integration: Use /link-tx to connect your FiveM server
  • Permissions: Ensure the bot has "Read Messages", "Send Messages", and "Embed Links" permissions
  • Role Setup: Create roles for managing who can use admin commands
  • Server Management: Use TX commands to manage players and server status

TXAdmin Setup

To connect your FiveM server:

  • Get your TXAdmin API key from your server panel
  • Use /link-tx with your server IP and API key
  • Bot will confirm connection
  • Use /tx-status to check server status
  • Use /tx-players to see online players
